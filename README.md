# Mission-to-Mars
## Project Overview
 This project is to demostrate the effectivness of HTML web scraping using Mongo DB, Bootstrap and Flask to build a locally hosted webpage that updates with live data from multiple online sources.
  
## Resources
  * Software: Pyhton 3.9.2 64 bit, Jupyter Notebook 6.1.4, Flask, Mongo DB, Bootstrap

## Results
This project was successful in retrieving facts about mars, current news articles related to the Mar's missions, and even high definition pictures of the red planet.  The website was also made mobile friendly.
